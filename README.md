## Development

Make sure you have [Node.js](https://nodejs.org) and [PNPM](https://pnpm.io) installed.

**Install the dependencies:**

```bash
pnpm install
```

**Then start the development server:**

```bash
pnpm dev
```

**To build the website:**

```bash
pnpm build
```
